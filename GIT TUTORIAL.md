## Перемещение между версиями
Для того, чтобы перемещаться между версиями файла необходимо использовать команду:

*__git checkout <commit_number>__*

В большинстве случаев достаточно использовать первые четыре символа из номера коммита.

После перехода между разными версиями для дальнейшей работы необходимо перейти в самое актуальное состояние. Для этого испольуйте команду:

*__git checkout master__*
